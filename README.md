# Character Database

A simple **React** project that serves as a database for characters.  
It allows writers and creators to store character cards with basic details in a clean and minimal interface.  

## 🚧 Project Status
This project is still under development.  
The core features are in place, but improvements and new functionality will be added over time.  

## ✨ Features (planned)
- Create and edit character cards  
- Store essential character information  
- Search and filtering options  
- Clean and minimal design
